底层原理

docker是一个client—server结构的系统
docker的守护进程运行在主机上
通过Socket从客户端访问
DockerServer接收到Docker-Client的指令就会执行这个命令！

Docker为什么比VM快
1.docker有比虚拟机更少的抽象层
2.docler用的是宿主机的内核，vm需要的是Guest OS

所以说docker不需要重新加载操作系统内核，避免引导。
